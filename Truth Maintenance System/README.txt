Compilation:
============
1. Move files "TMS.java" and "TMSInput.txt" from Assign4_KrishnaAgarwala.zip to the vcl.
2. If you have another set of input(s), please have the input placed into the TMSInput.txt file.
3. Once inside the same folder(having the "TMS.java" and "TMSInput.txt" files), compile the file by the command:

	javac TMS.java
	java TMS


Limitations:
1. The code does not entertain braces in the input. Please dont have braces in the input. Thanks.
